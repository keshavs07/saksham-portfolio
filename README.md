# Saksham - Photography Portfolio

A professional, elegant, and responsive photography portfolio built with React, TypeScript, and Vite. Features a dynamic image gallery and smooth cinematic UI effects.

## System Requirements

To run and edit this project locally, you will need:
- **Node.js** (v16.0 or higher)
- **npm** (Node Package Manager)

## Local Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/keshavs07/sakaham-portfolio.git
   cd sakaham-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and visit the local link provided in your terminal (usually `http://localhost:5173`).
